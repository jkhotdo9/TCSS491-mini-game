Missile Dodge (Web Mini Game)

How to run:
1) Unzip the folder.
2) Open index.html in Chrome/Edge/Firefox.
3) Click the canvas once if keys do not respond (it gives the canvas focus).

Controls:
- Move: WASD or Arrow Keys
- Restart: R

Game rules:
- Survive as long as possible.
- Score increases with time.
- Meteorioids spawn from all sides and get more frequent/faster over time.
